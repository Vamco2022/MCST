### 安装
实际上的话目前还在测试阶段，我只能发布.jar文件。如果你想尝试，那么请到甲骨文官网下载JDK21并安装后运行。

后面进入正式版后我会连着java一起打包，就不用下载java再运行了，直接就是.exe文件。

### 开发环境需求
+ logback以及依赖
+ org.json
+ okhttp3
+ JavaFX
+ JDK21
+ Flatlaf

没有这些程序会报错
